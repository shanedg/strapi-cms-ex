## install
NOTE: require node 10.x.x

`nvm install 10.13.0 && nvm use 10.13.0`

optionally, `nvm alias default 10.13.0`

`npm install strapi@alpha -g`

`strapi new project-name`, defaults are fine

`cd project-name`
